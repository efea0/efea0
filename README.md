# Hey, I'm Efe 👋

I'm a 19-year-old entrepreneur and product builder from Istanbul, Türkiye.

I'm currently building **Netik** — an AI-native media startup helping people understand news beyond headlines.

## 🚀 What I'm Building

**Netik** is my attempt to rethink how news is consumed in Türkiye.

I started building it because I believe people deserve a smarter, faster, and more honest way to follow what is happening in the world.

Most news products are built around speed, volume, and attention.  
Netik is being built around context, trust, and meaning.

The product is still in development, but I'm moving it forward step by step — by researching, building, testing, and learning.

## 🧠 How I Work

I learn by doing.

I've never waited for the perfect course, the perfect school, or the perfect moment. When a problem gets stuck in my head, I start working on it — wherever I am, with whatever I have.

That is how Netik started.

I care about understanding problems deeply before trying to solve them. I spend a lot of time researching markets, user behavior, media trends, product strategy, and how technology can create better experiences.

For me, building is not just writing code.  
It is connecting research, product thinking, design, technology, and execution.
